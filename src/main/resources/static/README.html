🚀 如何启动

方式一:直接打开(简单测试)

  1. 确保后端服务已启动 (端口 8080)
  2. 双击 frontend/index.html 直接在浏览器打开
  3. 或使用Live Server插件(推荐)

方式二:使用Python简易服务器

# 在 frontend 目录下执行
cd frontend
python -m http.server 3000

# 然后访问 http://localhost:3000

方式三:使用Nginx(生产环境)

# 将 frontend 目录放到 Nginx 的 html 目录
# 配置 Nginx 指向该目录即可

📝 使用指南

1. 注册账号

  1. 打开应用，点击“立即注册”
  2. 填写用户名、昵称（可选）、密码
  3. 点击注册按钮
  4. 记住你的用户ID（注册成功后会显示）

2. 添加好友

  1. 登录后，在左侧边栏找到“➕ 添加好友”按钮
  2. 点击按钮，弹出添加好友窗口
  3. 输入对方的用户ID
  4. 可以填写备注名称（可选）
  5. 点击“确认添加”
  6. 添加成功后，好友会出现在好友列表中
💡 提示：如何获取对方ID？
可以让对方登录后查看个人信息，或者通过后端API查询用户列表。

3. 开始聊天

  1. 在好友列表中点击好友头像或名称
  2. 右侧会打开聊天窗口
  3. 在底部输入框输入消息
  4. 按 Enter 键或点击“发送”按钮发送
  5. Shift + Enter 可以换行

🐛 常见问题

1. 跨域问题 (CORS)

如果看到跨域错误,请确保后端的 CorsConfig.java 已正确配置。

2. WebSocket连接失败

检查后端WebSocket端点是否正确,默认路径: /ws/chat/{userId}

3. 消息发送失败

确认后端API接口是否正常运行,查看浏览器控制台错误信息
