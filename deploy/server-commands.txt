# Fengbushi Server 101 Start Commands

## First Time Setup (After System Restart)
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

## Start Service
cd C:\fengbushi
.\start-backend.ps1

## Stop Service
.\stop-backend.ps1

## Check Status
.\check-backend.ps1

## View Logs
Get-Content C:\fengbushi\logs\fengbushi-server.log -Tail 50
