# Fengbushi Server 102 Start Commands

## First Time Setup (After System Restart)
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

## Start Service
cd C:\fengbushi
.\start-backend.ps1 server102

## Stop Service
.\stop-backend.ps1 server102

## Check Status
.\check-backend.ps1 server102

## View Logs
Get-Content C:\fengbushi\logs\fengbushi-server102.log -Tail 50
